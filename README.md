# 谜屿生日礼物｜来自世界的回响

## GitHub Pages 使用方式

把本资料夹内容上传到 GitHub Repository 根目录，然后开启 Pages。

建议结构：

```text
index.html
data/blessings.json
audio/
avatars/
images/
```

## 更新祝福

只需要修改 `data/blessings.json`。

```json
{
  "name": "布丁",
  "city": "台北",
  "avatar": "avatars/pudding.jpg",
  "audio": "audio/pudding.mp3",
  "wish": "主包生日快乐~要每天开开心心健健康康的呀(=^▽^=)",
  "poem": "希望月亮记得替我们照顾你，让你每天都能开开心心、健健康康地唱着喜欢的歌。"
}
```

## 素材命名建议

- 音频：`audio/name.mp3`
- 头像：`avatars/name.jpg`
- 文件名建议使用英文、数字、横线，不要用空格。
